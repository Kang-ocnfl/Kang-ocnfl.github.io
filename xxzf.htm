一．下载酷Q

插件依赖于酷Q框架运行，首先需要在酷Q社区（https://cqp.cc）下载一个酷Q，酷Q分Air和Pro两个版本，其中Air为免费版，Pro为付费版，对应功能也有所不同。Air版下载链接：https://cqp.cc/t/23253；Pro版下载链接：https://cqp.cc/t/14901。注意：请先解压酷Q压缩包再使用！

* 酷Q是一款智能机器人软件，严禁将酷Q用于违法用途（如发送广告/群发/诈骗、色情、政治等内容）。

二．下载插件

社区帖子地址：https://cqp.cc/t/48964（随时可能变动，建议加入QQ交流群638604615获取最新版本）下载完插件（cn.minedawn.xxzf.cpk）后，将其放置入酷Q目录\app文件夹中，然后运行酷Q，右键悬浮窗->应用->应用管理，从插件列表中选择消息转发并启用即可。

五．指令

主人命令(私聊机器人)：

1.     私聊 QQ号 消息 -可以让机器人给指定人发消息

2.     群聊 群号 消息 -可以让机器人在指定群发消息

3.     撤回 群转群/群监控/个人转群 序号 -撤回对应消息，序号需在转发格式处填写

4.     撤回关键词 [关键词] -按关键词撤回

5.     开启/关闭 群转群/群监控/个人转群(不填默认群转群) -一键开关转发

注：撤回功能需酷Q Pro版本

六．常见疑问

Q：为什么设置好了的配置，重启全没了？

A：先检查是否解压了酷Q压缩包再使用，关闭杀毒软件再尝试。

Q：为什么我按上述说明配置完成后还是无法转发？

A：请先检查配置，确认转发条件、关键词是否符合。若均无误，请查看酷Q日志（右键酷Q悬浮窗->日志）若日志中有绿色日志、显示消息发送成功，而群里看不到消息的话，说明您的机器人被腾讯屏蔽了，与酷Q和插件均无关。一般在您 异地登录、发送违规消息、频繁发送消息 后容易被腾讯屏蔽。解决方法有两种：

1.     先关闭插件，保持酷Q在线并等待一至两天，屏蔽会自动解除

2.     在机器人所在的电脑上使用浏览器打开QQ安全中心（aq.qq.com），并使用账号密码登录被屏蔽的账号，屏蔽就会立即解除（注：此方法成功率高，但部分情况下会失败）

Q：为什么我的插件崩溃了？

A：消息转发插件运营维护已有一段时间，几乎不存在BUG了，大部分插件崩溃均为用户配置错误导致，已知的因用户配置错误导致的崩溃有：

1.     关键词、敏感词中使用CQ码 且 左右括号未转义。解决方法：

a)      找到含有CQ码的转发规则，右键编辑。

b)      在所有括号（含“[ ] { } ( )”）的前面均加上一个“\”例如图片请填写“\[CQ:image”，语音请填写“\[CQ:record”，其他CQ码均参照此格式。

c)      保存配置即可

Q：我是老用户，我的授权是注册码，我该如何使用插件？

A：对于所有老用户，更新插件均不额外收费，授权为注册码的用户请右键酷Q悬浮窗->黎明消息转发->激活->旧版验证 并按照提示输入注册码。若您的机器码一栏为空，请关闭酷Q并使用管理员身份重新运行酷Q。

Q：为什么机器人转发不了图片？

A：转发图片、语音，撤回消息是酷Q Pro的专属功能，酷Q Pro收费为酷Q团队官方收费，与插件作者无关。
